**TELECOM AGENT PERFORMANCE ANALYSIS**

Acronymn Breakdown
1. MTD GA -->> Month to Date Gross Ads
2. AVG GA -->> Average Daily Gross Ads
3. MSISDN -->> Mobile Station International Subscriber Directory Number (Mobile Number)
4. ARPU -->> Average Revenue Per User
5. QA -->> Quality Gross Adds

Terminology Breakdown
1. MTD GA -->> Refers to subscriber addition from beginning of the month to date.
2. AVG GA -->> Dax formula calculates the total number of agent numbers partaking in gross ads for the month by the number of days.
4. ARPU -->> Distinct count the table with MSISDN and divide by the summation of total amount involed per row.
5. QA -->> A column in our table indicates when a subscriber consumes a package of 5euros and above.
6. Projection -->> Summation of active agents/ subscribers, divided by the current number of days, multiplied by the number of days in the month. This will give you a projection.
